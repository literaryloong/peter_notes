## 使用步骤

安装tyarn (官方推荐)

```
npm install yarn tyarn -g
```

初始化

```sh
tyarn create umi myapp
```

安装依赖

```sh
cd myapp && tyarn
```

开启 Umi UI（可选）：

```sh
tyarn add @umijs/preset-ui -D
```



使用cnpm

```shell
npm install -g cnpm --registry=https://registry.npm.taobao.org

```
